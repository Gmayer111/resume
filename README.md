## Table of contents

- [Installation](#installation)

## Installation

1. CLone the repository

```bash
 git clone https://github.com/yourusername/passion-bd-api.git
```

2. Install dependencies:

```bash
 yarn install
```

## Usage

To run the project, use the following command:

```bash
yarn dev
```
